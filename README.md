# 富邦 VCCT 月配息自動計算器

> 富邦人壽真豐利變額年金保險（VCCT）月配息自動計算器

## 功能
- 🌐 自動從富邦網站爬取基金單位淨值與每單位分配金額
- 🧮 自動計算每月/每年配息金額與年化報酬率
- 🔄 若自動爬取失敗，切換至手動輸入模式

## 計算公式
```
實際投入保費 = 保費 × (1 - 費用率)
換算美金     = 實際投入保費 ÷ 美金匯率
可購買單位數 = 換算美金 ÷ 基金單位淨值
每月配息(USD) = 單位數 × 每單位分配金額
每月配息(TWD) = 每月配息(USD) × 美金匯率
年化報酬率   = 每年配息(TWD) ÷ 實際投入保費 × 100%
```

## 安裝

```bash
pip install -r requirements.txt
```

## 使用方式

### 1. 修改設定（程式頂部）
```python
PREMIUM   = 1_000_000   # 保費（台幣）
FEE_RATE  = 0.05        # 保費費用率（5% = 0.05）
USD_RATE  = 31.4        # 美金匯率
```

### 2. 執行程式
```bash
python fubon_vcct_calculator.py
```
